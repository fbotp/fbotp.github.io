定义指针

int *p，对他初始化时，将其他变量的地址赋给它，如int *p=&n，即让p指向n，指针的变量名是p而不是*p，其他赋值操作可以使用p=&n而不是*p=&n，*p是用来取该地址对应的数值的
因为有了指针，在交换数据时可以不用交换数据的值，而是交换指针，而且甚至可以不用定义第三方指针，直接使用p1=&a，p2=&b的方法
指针数组：定义一个数组，把&a[0]赋给int指针(等同于 指针=a 的操作)，如果两个指针都指向同意数组中的元素时，可以进行相减操作，结果表示这两个元素间有几个元素。对指针数组的取值：下标法或者*(p+i)，a[i]等同于*(a+i)。在scanf数据时，也可以scanf(“%d”, p++)的做法
*p++由于*、++同优先级，自右向左结合，故等同于*(p++)，即先*p，然后将p++
使用数组做参数可以修改/交换数组元素的值
对于多维数组，如a[2][3]，a表示第一行元素的第一个元素的起始位置，a+1表示第二行元素的第一个元素的起始位置，*(*(a+1)+1)，a+1是第二行的起始地址，取*是该行0号元素的地址(不能省，虽然值一样，但含义不一样，如果不取，它不认识)，再+1是1号元素地址，再*则取出了第二行1号元素的值。

int *p[10] 指的指针数组 他是个数组 有10个元素 每个元素是int*类型,也就是指向int的指针 也就是存放10int指针变量,int (*p)[10] 指的数组指针 他是指针 指向一个含有10个元素的数组
c中没有字符串变量

指向函数的指针与指向数组的指针类似，int (*p)(int，int)，表示它可以指向函数返回值为整数且有两个int参数的函数，之后即可采用(*p)(a,b)的方法调用函数，函数指针可以用来传参调用

malloc函数返回值是指针，所以动态分配内存时要强制转换类型(int *)，即void *表示无指向的地址
int *p;p=NULL;合法，malloc完的要free

C中FILE是结构体，操作文件时需要*FILE，fopen打开文件指针，fclose关闭文件指针，fputc一个一个char存入文件(=putc)，fgetc一个一个char读取(=getc)，feof判断是否到了结尾(文件结束符的标志)，EOF=-1，fgets(目标字符数组，个数，文件指针)，fputs(要写入的字符数组，文件指针)，fprintf(文件指针，格式字符串，要写入文件的数据)格式化输出，fscanf(文件指针，格式字符串，要将数据写入的对象)，fread、fwrite是二进制方式操作，格式均为(buffer(要将数据存入/读取的地址)，字节数，要总共多少倍的字节数，文件指针)，rewind使文件指针再到开头，fseek(文件指针，位移量(距离起始点的字节数，用long类型)，起始点(SEEK_SET:0，开头，SEEK_CUR:1，当前位置，SEEK_END:2，结尾))，用rewind和fseek即可随机读写文件；ferror可检查是否有误(每执行一次输入/输出都会变化)，clearerr使文件出错标志和结束标志都置为0
