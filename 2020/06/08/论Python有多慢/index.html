论Python有多慢

发表于 2020-06-08 更新于 2020-06-11 分类于 Programming
漫长的等待。

Python简单归简单，实在是太太太太耗内存了，数据一大就不行了。以后做大数据的时候，还是能用Java/Spark就用吧，速度是硬伤，不会还是要好好学。:(
在Linux终端下使用matplotlib，数据一大，运行就越来越慢慢慢慢慢，忍不了了，果然没有好馅饼，简单是简单，慢更慢了，和IDLE有的一拼。
论什么时候才会放弃Python迎接Java/Spark。

是我憨憨了，根据网上大神的说法，尽量减少plt.scatter/plot之类的这种函数的调用，就不会出那样的问题，学的还是太少了。

# Python # matplotlib # idle
永远失去了猪窝
C语言准备就绪！
