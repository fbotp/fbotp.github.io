第三章

类型 short int int long int long long int char
长度/字节 2 2/4 4 8 1
还有float double complex
%m.nf：包括负号、整数部分、小数点、小数部分总长m，小数部分长n
%m.ne：包括负号、整数部分、+/-e、指数部分长共m，指数部分长n
%g：从%f %e中挑一个短的输出
printf()中%E %X %G可大写，对应的E、X大写，其他均不可
要输出%时要printf(%%)
使用getchar()可以获得类似回车换行之类的无法在屏幕上显示的字符

第四章

关系运算符优先级(从高到低)
逻辑运算符：与或非(&!|)
!->算数运算符->关系运算符(大于小于)->&/|->条件运算符(if/?)->赋值运算符
例如
!a&&b||x>y&&c <==> (!a)&&b||(x>y)&&c
bool等价于_Bool

第五章

do...while();与while()区别就在前者无论while成立不成立都会执行一次

strcpy(目的，源)，strncp(目的，源，前n个)，strlen(str)即到’\0’为止的长度(不包含’\0’)
